Staple Stable Motion Typography

project description

This project is the second time I've worked with After Effects, started because I really enjoyed my experience learning it in high school, wanting to develop those skills and learn more about the application. As my first personal project in motion design, I wanted to use a soong from my favourite anime and use that enthusiasm to really delve in and push myself while also seeing what I am capable of, restricted by my laptop. This made for a very satisfying project that made me want to keep going with the next chance I got.
